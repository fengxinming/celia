## Installation

### Load `celia` via classical `<script>` tag

```html
<script src="https://cdn.jsdelivr.net/npm/celia@6.0.0-beta.2/iife.min.js"></script>
```

### CommonJS style with npm

```bash
npm install celia --save
```
